Basic Concepts about Flexible layout webpages

The flexible box layout module was designed as a one-dimensional layout model, and as a method that could offer space distribution between items in an interface and powerful alignment capabilities.

When we describe flexbox as being one-dimensional we are describing the fact that flexbox deals with layout in one dimension at a time — either as a row or as a column. This can be contrasted with the two-dimensional model of CSS Grid Layout, which controls columns and rows together.

The two axes of flexbox

When working with flexbox you need to think in terms of two axes — the main axis and the cross axis. The main axis is defined by the flex-direction property, and the cross axis runs perpendicular to it. Everything we do with flexbox refers back to these axes, so it is worth understanding how they work from the outset.

CSS Flexbox Properties:

The following table lists all the CSS Flexbox Container properties:

Property: Description:
align-content Modifies the behavior of the flex-wrap property. It is similar to align-items, but instead of aligning flex items, it aligns flex lines
align-items Vertically aligns the flex items when the items do not use all available space on the cross-axis
display Specifies the type of box used for an HTML element
flex-direction Specifies the direction of the flexible items inside a flex container
flex-flow A shorthand property for flex-direction and flex-wrap
flex-wrap Specifies whether the flex items should wrap or not, if there is not enough room for them on one flex line
justify-content Horizontally aligns the flex items when the items do not use all available space on the main-axis

Flexbox Elements

A flex container with three flex items

                
Example:

                
< div class="flex-container" >
< div >1< /div >
< div >2< /div >
< div >3< /div >
< div >4< /div >
< /div >

The flex container becomes flexible by setting the display property to flex: 

                
Example:

                
.flex-container {
    display: flex;
}
                
                
Output:

The main flex properties are:

  1. flex-direction;
  2. flex-wrap;
  3. flex-flow;
  4. justify-content;
  5. align-items;
  6. align-content;

1) Flex-direction property:

The flex-direction property defines in which direction the container wants to stack the flex items.

                    
Example:

                    
The column value stacks 
 the flex items vertically :

                    
.flex-container {
    display: flex;
    flex-direction: column;
}
                    
                    
Output:

                    
                

                    
Example:

                    
The row value stacks 
 the flex items horizontally

                    
.flex-container {
    display: flex;
    flex-direction: row;
}
                    
                    
Output:

                    
                

                    
Example:

                    
The column-reverse value stacks 
 the flex items vertically:

                    
.flex-container {
    display: flex;
    flex-direction: column-reverse;
}
                    
                    
Output:

                    
                

                    
Example:

                    
 Row-reverse value stacks 
 from right to left horizontally:

                    
.flex-container {
    display: flex;
    flex-direction: row-reverse;
}
                    
                    
Output:

2) Flex-wrap Property:

The flex-wrap property specifies whether
the flex items should wrap or not.

                    
Example:

                    
The wrap value specifies that 
 the flex items will wrap :

                    
.flex-container {
    display: flex;
    flex-wrap: wrap;
}
                    
                    
Output:

                    

                

                    
Example:

                    
The nowrap value specifies that 
 the flex items will not wrap

                    
.flex-container {
    display: flex;
    flex-wrap: nowrap;
}
                    
                    
Output: 

                    

                
                    
Example:

                    
.flex-container {
    display: flex;
    flex-wrap: wrap-reverse;
}
                    
                    
Output:

3) Flex-flow property:

The flex-flow property is a shorthand property for setting both the flex-direction and flex-wrap properties. 

                    
Example:

                    
.flex-container {
    display: flex;
    flex-flow: row;
}
                    
                    
Output:

4) Justify-content property:

The justify-content property is used to align the flex items:

                    
Example:

                    
The center value align the flex items at the center

                    
.flex-container {
    display: flex;
    justify-content: center;
}
                    
                    
Output:

                    

                

                    
Example:

                    
The  flex-start value aligns  items at the beginning 
of the container(default)

                    
.flex-container {
    display: flex;
    justify-content: flex-start;
}
                    
                    
Output:

                    

                

                    
Example:

                    
The flex-end value align the flex items  at the end 
of the container:

                    
.flex-container {
    display: flex;
    justify-content: flex-end;
 }
                    
                    
Output:

5) The Align-items property:

The align-items property is used to align the flex items.

                    
Example:

                    
The center value align items  in the middle 
 of the container:

                    
 .flex-container {
    display: flex;
    height: 200px;
    align-items: center;
}
                    
                    
Output:

                    
                

                    
Example:

                    
The flex-start  value aligns the flex items 
 at the top of the container:

                    
.flex-container {
    display: flex;
    height: 200px;
    align-items: flex-start;
}
                    
                    
Output:

                    
                

                   
Example:

                   
The flex-end value aligns the flex items 
 at the bottom of the container:

                   
.flex-container {
    display: flex;
    height: 200px;
    align-items: flex-end;
}

6) Align-content property:

The align-content property is used to align the flex lines.

                       
Example:

                       
The space-between value displays the flex lines 
 with equal space between them:

                       
.flex-container {
    display: flex;
    height: 600px;
    flex-wrap: wrap;
    align-content: space-between;
}
                       
                       
Output:

                       
                    

                     
Example:

                     
The stretchvalue stretches the flex lines to take up 
 the remaining space (this is default):

                     
.flex-container {
    display: flex;
    height: 600px;
    flex-wrap: wrap;
    align-content: stretch;
}
