<html lang="en"> The website is in english.
<meta charset="UTF-8"> This tag is used to specify the character's codification
<title>Title...</title> The title between the tags is the title that will appear in the browser
<meta name="viewport" content="width=device-width, initial-scale=1.0"> This tag makes adjust the size of the page on any screen of any device
<meta name="description" content="..."> You can add meta name description in the head if you want to give a simple description of the website
<script></script> This tag lets connect to a javascript sheet
<link rel="stylesheet" href="style.css"> This tag lets connect to a CSS sheet
<meta name="keywords" content="..."> This tag lets you add keywords
