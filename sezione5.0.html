Basic Html

  • HTML is the standard markup language for Web pages and with it you can create your own website

  • HTML uses "markup" to annotate text, images, and other content for display in a Web browser. HTML markup includes special "elements" such as:
    < head >, < title >, < body> , < header >, < footer >, < article >, < section >, < p >, < div >, < span >, < img >, < video >, < ul >, < ol >, < li > and many others.

  • The < !DOCTYPE html> declaration defines that this document is an HTML5 document and The < html > element is the root element of an HTML page

  • The < head > element contains meta information about the HTML page for example: description (we use < meta name= "description" content= "...">), author (we use < meta name ="author" content="..." ),the website's language(for an english website we use < html lang= "en">), characters coding (we use < meta charset="UTF-8"> for example), the title of the website is what you see on the browser(< title >...< /title >)...

  • The < body > element defines the document's body, and is a container for all the visible contents, such as: headings(< h1 >Title< /h1 > for big titles,< h2>Title< /h2>< h3>Title< /h3> till < h6>Title< /h6> for more little titles than < h1>Title< /h1>), paragraphs(< p >text...< /p >), images(< img src="link..." alt="image description(optional)), hyperlinks(< a href="URL...">hyperlink text< /a>), tables(< table >< /table >), lists(< ul>< /ul> for unordered lists and < ol >< /ol > for ordered lists), ...
