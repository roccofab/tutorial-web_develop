Html contains several elements for defining text with a special meaning:

  • < b > -Bold text
  • < strong > -Important text
  • < em > -Emphasized text
  • < mark > -Marked text
  • < small > -Smaller text
  • < i > -Italic text
  • < del > -Deleted text
  • < sub > -Subscript-text
  • < sup > -Superscript text

Example:



                      
< html lang="en" >
< head >
  < meta charset="UTF-8">
  < meta name ="viewport" content="width=device-width", initial-scale="1.0">
  < title >Website< /title >
< /head >
< body >
  < h1 >My website< /h1 >
  < p >This is my website< /p >
  < p >< strong >This is my website< /strong >< /p >
  < p >< b >This is my website< /b >< /p >
  < p >< em >This is my website< /em >< /p >
  < p >This < sub >is my< /sub >website< /p >
  < p >This is my < small >website< /small >< /p 
< /body >
< /html >
